# Requirement

- Python 3.8+

# Feature

- Parse sql and create a syntax tree.
- Extract tables, columns, and functions.
- Controlling access to objects.
- Replace table, column expression.
- Rebuild SQL.
- Output the parsed syntax tree as JSON.
